# ポスナビ v0.7

## 主な変更
- 初期版を意識した、地図中心のシンプルなUI
- 「配布開始」で入力画面を出さず即開始
- 配布終了後、履歴からエリア・配布物・メモを整理する前提
- 通常タップは1部ずつ追加
- 前回部数を薄いグレーで表示
- マーカー長押し、または右下の「…」から詳細を表示
- 3秒間の取り消しは、その地点の直前の操作を元に戻す
- 詳細内の「今回分を全部消す」で、その地点の今回記録を0に戻す
- 左上の↺で今回の配布記録を全部削除
- 配布履歴を地図上に再表示
- 住所検索、現在地表示、住宅の事前登録
- 配布不可・要注意・複数部予定・会社・集合住宅に対応

## 更新方法
`index.html`、`style.css`、`app.js`、`README.md`をGitHubの`posnavi`フォルダへ上書きし、GitHub DesktopでCommit、Pushしてください。

## 保存について
データはブラウザのlocalStorageに保存されます。端末間同期はまだありません。
