# ポスナビ v0.4

## 今回追加した機能

- 配布開始・終了
- エリア名と配布物の記録
- 住宅マスター
- 配布不可・要注意・複数部予定
- 基本配布部数
- 住所・建物名検索
- 日付とエリア別の配布履歴
- 集合住宅や会社向けの複数部登録

## ファイル構成

- index.html
- style.css
- app.js

## GitHub Pagesへの反映

1. この3ファイルをリポジトリ直下に置く
2. GitHub Desktopで変更を確認
3. Summaryに `ポスナビ v0.4` と入力
4. Commit to main
5. Push origin
6. 1〜3分待って公開URLを再読み込み

## 注意

データはブラウザのlocalStorageに保存されます。
別のスマホや別のブラウザには自動同期されません。
