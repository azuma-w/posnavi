# ポスナビ v0.6

ポスティングの配布地点、住宅情報、配布履歴を地図上で管理する試作版です。

## v0.6 の主な変更

- 通常タップは常に1部ずつ加算
- 前回の配布部数を薄いグレーで表示
- 前回部数の表示はタップできない仕様
- マーカー右側の「…」から詳細を表示
- マーカー長押しでも詳細を表示
- 配布後3秒間だけ「取り消す」ボタンを表示
- 上下が切れにくい固定レイアウトへ変更
- 詳細画面から住宅情報・履歴・編集を確認

## 公開手順

1. `index.html`、`style.css`、`app.js`、`README.md` をGitHubの `posnavi` フォルダへ上書きします。
2. GitHub Desktopでコミットします。
3. `Push origin` を押します。
4. GitHub Pagesへ反映後、スマートフォンで再読み込みします。

## 注意

データはブラウザのLocalStorageに保存されます。端末間の同期やバックアップ機能はまだありません。
